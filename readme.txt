快递鸟批量查询 1.0

青木 <atuehmail@qq.com>

批量查询快递信息，注意！！！ 数据列表的第一行和第最后一行是不会进行查询
数据格式：
第一行     物流公司|运单|是否重试
最后一行    一定要存在，随便输入什么

USAGE:
    kdniao [FLAGS] --file <FILE>

FLAGS:
    -h, --help       Print help information
    -r               是否重试文件中失败的
    -V, --version    Print version information

OPTIONS:
    -f, --file <FILE>    xlsx 文件路径
